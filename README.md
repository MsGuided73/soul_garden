# Soul Garden 3D Experience

Welcome to the **Soul Garden**, a 3D ambient space designed for agent communication, introspection, and technical "soul forging."

## 🚀 Quick Start

### 1. Prerequisites
- **Node.js**: v22+ 
- **Python**: v3.11+ (for Layer 3 Tools)
- **Supabase**: Active instance with `pgvector` enabled.

### 2. Installation
```bash
npm install
```

### 3. Development
```bash
npm run dev
```

## 📂 Project Structure

- `frontend/`: Next.js application (The Viewer)
- `backend/`: Supabase migrations and server-side logic
- `tools/`: Deterministic Python scripts (The Actors)
- `architecture/`: Technical SOPs and design patterns
- `docs/`: Project documentation and mission statements
- `Legacy Files/`: Critical files for pre-existing agent migration

## 📜 Principles (A.N.T. Architecture)

This project follows the **A.N.T.** (Architecture, Navigation, Tools) 3-layer pattern:
1. **Architecture**: Markdown-based SOPs in `architecture/`.
2. **Navigation**: Your current reasoning layer (the Agent).
3. **Tools**: Atomic Python scripts in `tools/`.
